1. Proje Tanımı

Bu projede, nesneye yönelik programlama prensipleri (kalıtım, kapsülleme, çok biçimlilik) kullanılarak bir Banka Otomasyon Sistemi geliştirilmiştir. Sistem; müşteri yönetimi, banka hesabı işlemleri (vadesiz ve yatırım hesabı), kredi kartı işlemleri ve para transferi gibi temel bankacılık operasyonlarını desteklemektedir.

Kullanılan NYP prensipleri:

  • Kalıtım (Inheritance): Kisi → BankaPersoneli, Musteri | BankaHesabi → VadesizHesap, YatirimHesabi
  • Kapsülleme (Encapsulation): Tüm alanlar private, getter/setter metotları ile erişim
  • Çok biçimlilik (Polymorphism): toString() metotlarının override edilmesi, BankaHesabi referansı ile alt sınıf nesnelerine erişim
  • ArrayList kullanımı: Müşteri listesi, hesap listesi ve kredi kartı listesi

4. Sınıf Açıklamaları ve Kaynak Kodlar

4.1 Kisi.java (com.bank.app.people)

Tüm kişi türlerinin temel sınıfıdır. Ad, soyad, email ve telefon numarası alanlarını içerir.

public class Kisi {
    private String ad, soyad, email;
    private int telefonNumarasi;

    public Kisi(String ad, String soyad, String email, int telefonNumarasi) {
        this.ad = ad; this.soyad = soyad;
        this.email = email; this.telefonNumarasi = telefonNumarasi;
    }
    // Getter ve Setter metotlari + toString()
}

4.2 BankaPersoneli.java (com.bank.app.people)

Kisi sınıfından miras alır. personelID random olarak üretilir. Sorumlu olduğu müşterileri ArrayList ile saklar.

public class BankaPersoneli extends Kisi {
    private String personelID;  // Random uretilir: "P" + 6 hane
    private ArrayList<Musteri> musteriler;

    public BankaPersoneli(String ad, String soyad, String email, int tel) {
        super(ad, soyad, email, tel);
        this.personelID = "P" + (100000 + random.nextInt(900000));
        this.musteriler = new ArrayList<>();
    }
    public void musteriEkle(Musteri musteri) { musteriler.add(musteri); }
}

4.3 Musteri.java (com.bank.app.people)

Kisi sınıfından miras alır. Hesap ve kredi kartı yönetimi yapar. hesapSil() bakiye kontrolü, krediKartiSil() borç kontrolü yapar.

public class Musteri extends Kisi {
    private String musteriNumarasi;  // Random: "M" + 8 hane
    private ArrayList<BankaHesabi> hesaplar;
    private ArrayList<KrediKarti> krediKartlari;

    public void hesapEkle(String hesapTuru) {
        // "Vadesiz" ise VadesizHesap, "Yatirim" ise YatirimHesabi olusturur
    }
    public void hesapSil(BankaHesabi hesap) {
        if (hesap.getBakiye() > 0) // Uyari ver
        else hesaplar.remove(hesap); // Sil
    }
    public void krediKartiSil(KrediKarti kart) {
        if (kart.getGuncelBorc() > 0) // Uyari ver
        else krediKartlari.remove(kart); // Sil
    }
}

4.4 BankaHesabi.java (com.bank.app.accounts)

Tüm hesap türlerinin temel sınıfıdır. IBAN "TR" + 24 haneli random sayı olarak üretilir.

public class BankaHesabi {
    private String iban;   // Random: "TR" + 24 hane
    private double bakiye;

    public BankaHesabi(double bakiye) {
        StringBuilder ib = new StringBuilder("TR");
        for(int i=0; i<24; i++) ib.append(random.nextInt(10));
        this.iban = ib.toString();
        this.bakiye = bakiye;
    }
}

4.5 VadesizHesap.java (com.bank.app.accounts)

BankaHesabi'dan miras alır. Para transferi ve kredi kartı borç ödeme işlemlerini gerçekleştirir.

public class VadesizHesap extends BankaHesabi {
    public void paraTransferi(BankaHesabi alici, BankaHesabi gonderen, double miktar) {
        gonderen.setBakiye(gonderen.getBakiye() - miktar);
        alici.setBakiye(alici.getBakiye() + miktar);
    }
    public void krediKartiBorcOdeme(KrediKarti kart, double miktar) {
        setBakiye(getBakiye() - miktar);
        kart.setGuncelBorc(kart.getGuncelBorc() - miktar);
    }
}

4.6 YatirimHesabi.java (com.bank.app.accounts)

BankaHesabi'dan miras alır. Para ekleme ve para çekme işlemlerini destekler.

public class YatirimHesabi extends BankaHesabi {
    public void paraEkle(double miktar) { setBakiye(getBakiye() + miktar); }
    public void paraCek(double miktar) {
        if(getBakiye() >= miktar) setBakiye(getBakiye() - miktar);
    }
}

4.7 KrediKarti.java (com.bank.app.cards)

Kredi kartı bilgilerini yönetir. Kart numarası 16 haneli random sayı olarak üretilir. Borç değiştiğinde kullanılabilir limit otomatik güncellenir.

public class KrediKarti {
    private String kartNumarasi; // Random 16 hane
    private double limit, guncelBorc, kullanilabilirLimit;

    public KrediKarti(double limit, double guncelBorc) {
        this.kullanilabilirLimit = limit - guncelBorc;
    }
    public void setGuncelBorc(double borc) {
        this.guncelBorc = borc;
        this.kullanilabilirLimit = this.limit - borc; // Otomatik guncelle
    }
}

4.8 BankaService.java (com.bank.app.service)

İş mantığı katmanıdır. Tüm bankacılık operasyonlarını merkezi olarak yönetir: müşteri/personel oluşturma, hesap açma, para yatırma, transfer, kredi kartı tanımlama, borç ödeme, hesap/kart silme.
